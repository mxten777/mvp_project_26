**Resort BAIKAL**은 바이칼 호수의 자연과 휴식을 담은 **프리미엄 리조트 예약/관리 웹 애플리케이션**입니다.

🎯 **핵심 가치**
- **🏆 프리미엄 경험**: 최고급 UI/UX와 브랜드 일관성
- **📱 완전한 반응형**: 모바일/태블릿/데스크탑 모든 기기 최적화
- **⚡ 실시간 관리**: Firebase 기반 실시간 데이터 동기화
- **🎨 세련된 디자인**: 퍼플 테마 기반 현대적 디자인 시스템

## 🌟 주요 특징

### 🎨 **프리미엄 UI/UX**
- 세련된 퍼플 테마 (`purple-600` 기반)
- 프리미엄 타이포그래피 시스템 (Space Grotesk, Poppins, Inter Variable)
- 일관된 브랜딩과 그라데이션 효과
- 직관적이고 현대적인 사용자 인터페이스

### 📱 **완전한 모바일 최적화**
- 모바일 우선 반응형 디자인 (Mobile-First Responsive)
- 정밀한 햄버거 메뉴 네비게이션 (개선된 가독성)
- 터치 친화적 인터페이스 (44px 최소 터치 영역)
- ResponsiveUtils 시스템으로 디바이스별 최적화

### 🎯 **Progressive Web App (PWA)**
- 완전한 PWA 지원 (오프라인 접근, 설치 가능)
- Service Worker 기반 캐시 전략
- 모바일 앱과 같은 사용자 경험
- 홈화면 추가 및 스플래시 스크린

### ✨ **고급 애니메이션 시스템**
- Framer Motion 기반 부드러운 페이지 전환
- 마이크로 인터랙션 (hover, focus, click 효과)
- 로딩 상태 애니메이션 (스피너, 스켈레톤)
- 스크롤 기반 애니메이션 (Intersection Observer)

### 🌙 **다크 모드 지원**
- 완전한 다크/라이트 모드 토글
- 시스템 설정 감지 및 자동 전환
- 부드러운 테마 전환 애니메이션

```

  components/   # 공통 UI 컴포넌트(Button, Modal, Toast, Layout, ThemeToggle 등)

  index.css     # 글로벌 스타일 및 다크모드
  main.jsx      # 앱 진입점
```
